# Visual Studio Code 使用

### 快捷键

* Ctrl + F 查找
* Ctrl + H 替换
* Alt + ← 回到光标上一次的位置
* Ctrl + Shift + H 全局替换

### 扩展

* `GitLens` 直接在 VSCode 中使用 git
* `Vetur` Vue 工具
* `Kite` 人工智能自动补齐
* `C/C++ Compile Run` 更快的编译运行 C/C++ 程序
* `Markdown PDF` 将 md 文件转成 pdf，使用方法：先按 shift + ctrl + p，然后输入 `Markdown PDF:Export(pdf)`
